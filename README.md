# Wauw-logger


Clean logger engine for your Node.js projects

Possibles logging functions:

```js
log.info.ln('test');
log.debug.ln('test');
log.warn.ln('test');
log.error.ln('test');

log.info.ok('test');
log.debug.ok('test');
log.warn.ok('test');
log.error.ok('test');

log.info.fail('test');
log.debug.fail('test');
log.warn.fail('test');
log.error.fail('test');

log.info.promise('test');
log.debug.promise('test');
log.warn.promise('test');
log.error.promise('test');
```
